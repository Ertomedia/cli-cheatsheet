# Vercel

 Live demo: [https://manual.now.sh](https://manual.now.sh)

## Instalasi

```bash
npm i -g vercel
```

## Perintah umum

 Untuk bantuan ketik `vercel help`.

### Local server

```bash
vercel dev
```

### Mengaktifkan Vercel

Jalankan command berikut dalam folder project.

```bash
vercel init
```

### Deployment

Jalankan command berikut dalam folder project.

```bash
vercel
```

### Update project

Jalankan command berikut dalam folder project.

```bash
vercel --prod
```
