# Vue

![](.gitbook/assets/head_vuejs.jpg)

## Instalasi

### Via npm

```bash
npm install -g @vue/cli
```

### Via yarn

```bash
yarn global add @vue/cli
```

## Perintah Umum

### Membuat project baru

```bash
vue create my-project
```

Atau dengan GUI.

```bash
vue ui
```
