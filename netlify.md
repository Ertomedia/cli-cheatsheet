# Netlify

 Live demo: [https://climyid.netlify.com/](https://climyid.netlify.com/)

## Perintah umum

### Local server

```bash
netlify dev
```

### Deploy manual ke Netlify

Jalankan command berikut dalam folder project.

```bash
netlify sites:create --manual --with-ci
```
